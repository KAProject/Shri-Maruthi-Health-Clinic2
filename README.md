# Shri Maruthi Health Clinic

A responsive doctor's website built with HTML5, CSS3, and JavaScript featuring 8 pages with complete functionality.

## Pages Included

1. **Home (index.html)** - Landing page with carousel, clinic timings, and intro
2. **About Us (about.html)** - Doctor profile, education, achievements, and certifications
3. **Photos (photos.html)** - Gallery of clinic and doctor images/videos
4. **Certifications (certifications.html)** - 10+ professional certifications
5. **Our Team (team.html)** - Staff members with photos and descriptions
6. **Services (services.html)** - Services offered and price list table
7. **Reviews (reviews.html)** - Patient testimonials and ratings
8. **Location (location.html)** - Address details and embedded map
9. **Contact (contact.html)** - Contact information and contact form

## Features

- **Responsive Design** - Works on all devices (desktop, tablet, mobile)
- **Mobile Navigation** - Hamburger menu for mobile devices
- **Professional Layout** - Clean, medical-themed design
